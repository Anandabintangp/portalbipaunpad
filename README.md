# Portal BIPA Unpad - GitHub Pages Starter

Ini paket awal yang pasti bisa dibuka di GitHub Pages.

## Isi file

- `index.html` = halaman pilih portal
- `operator.html` = preview portal operator
- `pemelajar.html` = preview portal pemelajar
- `assets/style.css` = desain
- `assets/operator.js` = logika operator preview
- `assets/pemelajar.js` = logika pemelajar preview
- `config.js` = tempat URL API Apps Script nanti

## Cara upload ke GitHub

1. Buka repository `portalbipaunpad`
2. Upload semua isi folder ini ke root repository
3. Pastikan ada file `index.html` di root
4. Commit changes
5. Buka Settings > Pages
6. Source: Deploy from a branch
7. Branch: main
8. Folder: /root
9. Save
10. Buka:
   `https://anandabintangp.github.io/portalbipaunpad/`

## Catatan

Versi ini masih preview statis. Jangan masukkan password, nilai asli, atau data privat ke GitHub.
Koneksi ke Spreadsheet/Drive tetap lewat Apps Script API.
